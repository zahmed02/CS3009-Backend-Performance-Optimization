================================================================================
TROUBLESHOOTING
================================================================================
- If you see version conflict warnings (MSB3277), they are harmless and do not
  affect test execution. They arise because the main project uses EF Core 8.0.3
  while some dependencies pull 8.0.13; the build resolves to 8.0.3.
- If tests fail to compile, ensure the reference to the main project is correct.
- To clean and rebuild: dotnet clean && dotnet build
